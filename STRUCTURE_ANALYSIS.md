#### 2️⃣ 結帳頁面缺失

**問題描述：**

- 購物流程：瀏覽商品 → 購物車 → **❌ 結帳確認** ❌ → 訂單完成
- Cart.jsx 可以提交訂單，但沒有結帳確認頁面
- 完整流程中斷

**影響：** 🟡 **需要補充，完成主流程**

**解決方案：**
創建 `Checkout.jsx` 頁面：

```
1. 訂單摘要展示
2. 收貨人信息確認
3. 支付方式選擇
4. 訂單總額確認
5. 完成訂單提交
```

### Phase 2：結帳流程完整化

#### 步驟 1：創建 Checkout.jsx 頁面

**文件位置：** `src/views/Checkout.jsx`

**功能需求：**

```jsx
function Checkout() {
  // 1. 顯示購物車摘要
  //    - 所有商品列表
  //    - 小計、稅金、運費、總額
  //
  // 2. 展示收貨人信息（從 Cart 表單傳入）
  //    - 姓名、電話、地址、備註
  //
  // 3. 支付方式選擇
  //    - 信用卡
  //    - 便利超商取貨付款
  //    - 轉帳
  //
  // 4. 訂單確認
  //    - 再次檢查所有信息
  //    - "返回修改"按鈕
  //    - "確認下單"按鈕
  //
  // 5. 訂單完成提示
  //    - 顯示訂單號碼
  //    - 預期送達時間
  //    - 返回首頁按鈕
}
```

#### 步驟 2：更新路由配置

**文件：** `src/routes/index.jsx`

```jsx
// 新增路由
{ path: "checkout", element: <Checkout /> }

// 完整路由結構
const routes = [
  {
    path: "/",
    element: <Layout />,
    children: [
      { index: true, element: <Home /> },
      { path: "products", element: <Products /> },
      { path: "product/:id", element: <Product /> },
      { path: "cart", element: <Cart /> },
      { path: "checkout", element: <Checkout /> },  // ✅ 新增
      { path: "*", element: <NotFound /> },
    ],
  },
  { path: "login", element: <AdminLogin /> },
  { path: "admin/product", element: <AdminProduct /> },
];
```
